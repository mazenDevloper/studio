# **App Name**: DriveCast

## Core Features:

- Dashboard Widgets: Display widgets with information from Prayer Array, NASA Moon Library, Google Maps, YouTube Saved List Bar, and a personalized YouTube Suggestion Bar. The YouTube suggestions will depend on the channels added in the 'Media' screen.
- Media Screen: A screen to manage the user's favorite YouTube channels and when click specific channel from this grid show me result of newest video in its channel. Includes a search bar to find new channels and search videos and a button to add them to the list.
- Personalized YouTube Suggestions: AI-powered feature that uses a tool to learn user preferences from selected YouTube channels and provide personalized content suggestions on the dashboard.

## Style Guidelines:

- Primary color: Deep purple (#6A4EA3) to capture a futuristic and sleek aesthetic suitable for in-car tech.
- Background color: Dark purple (#1A132F) to provide high contrast with interface elements and reduce eye strain in a dark environment.
- Accent color: Teal (#41B883) to highlight key interactive elements and provide a visual pop.
- Body font: 'Inter', a sans-serif font, will be used to create a modern, sleek and neutral reading experience.
- Headline font: 'Space Grotesk', a sans-serif font, will be used to create a computerized, techy, scientific feel.
- Use minimalist and modern icons suitable for a car interface, ensuring they are easily recognizable at a glance.
- The layout should follow a grid system, similar to the reference image, with rounded corners for all widgets and interface elements to provide a soft, modern feel.
